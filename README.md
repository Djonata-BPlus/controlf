# CONTROLF Extension


## Como usar

1. Pressione `Ctrl+Shift+P` no VS Code e digite `controlf: Buscar arquivos no servidor`
![Passo 1](https://github.com/Djonata-BPlus/controlf/blob/main/assets/step-1.png)

2. Na primeira vez que tentar acessar a tela de busca ele vai redirecionar para a configuração de servidores, e lá você deve adicionar um servidor para poder começar a bsucar.
![Passo 2](https://github.com/Djonata-BPlus/controlf/blob/main/assets/step-2.png)
